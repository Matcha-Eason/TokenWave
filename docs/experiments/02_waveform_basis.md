# 多通道波形基实验

本实验对应当前方案的前两步：

1. 三通道清洗与固定长度重采样。
2. 使用非负矩阵分解（NMF）学习可解释波形基，并做初步前缀预测验证。

## 输入

输入文件：

```text
outputs/coderforge_full/turn_timeseries.parquet
```

使用三条主信号：

| 中文名 | 字段 | 含义 |
|---|---|---|
| 模型输出波 | `assistant_tokens` | 每轮 AI 回复 token 数 |
| 思考规划波 | `thinking_tokens_est` | 每轮 think 内容 token 估计 |
| 环境反馈波 | `user_tokens_est` | 前一轮 user/tool response token 估计，大多是工具返回 |

事件信号暂不参与 NMF 主波形基学习，只保留作后续解释：

| 中文名 | 字段 |
|---|---|
| 编辑事件 | `edit_count` |
| 测试事件 | `test_count` |
| 错误反馈事件 | `error_feedback_count` |
| 结束事件 | `finish_call_count` |

## 方法

每条轨迹被重采样到 64 个归一化进度点。信号先做：

```text
log(1 + 原始值)
按全局通道 75 分位数缩放
按单条轨迹每个通道 95 分位数归一
```

这样做的目的不是保留绝对 token 规模，而是学习“形状”：什么时候出现峰、哪个通道在什么阶段抬升、多个波形如何叠加。

NMF 学习目标：

```text
每条轨迹的三通道波形 ≈ 若干个非负波形基的加权和
```

正式实验参数：

```bash
MPLCONFIGDIR=.matplotlib \
/Users/gyc/.cache/codex-runtimes/codex-primary-runtime/dependencies/python/bin/python3 \
scripts/waveform_basis_experiment.py \
  --input outputs/coderforge_full/turn_timeseries.parquet \
  --out-dir outputs/waveform_basis_experiment \
  --max-trajectories 30000 \
  --components 8 \
  --points 64
```

## 结果

正式实验使用 30,000 条轨迹：

```text
训练集：22,500
测试集：7,500
重采样点数：64
NMF 波形基数量：8
```

完整曲线重构结果：

| split | normalized_mse | r2 |
|---|---:|---:|
| train | 0.3007 | 0.1033 |
| test | 0.3043 | 0.1005 |

解释：8 个非负波形基能把三通道轨迹形状压缩到明显低于原始方差的误差水平。`R2` 不高，说明轨迹个体差异很大；但 `normalized_mse = 0.3043` 说明它确实学到了稳定的公共形状，不是完全随机。

前缀预测后续曲线结果：

| 前缀比例 | 平均模板 normalized_mse | 无监督 NMF 前缀拟合 | 前缀到 NMF 权重的岭回归 |
|---:|---:|---:|---:|
| 25% | 0.4554 | 0.7517 | 0.4520 |
| 50% | 0.3627 | 0.4747 | 0.3526 |
| 75% | 0.3143 | 0.3450 | 0.3186 |

解释：

- 纯无监督前缀拟合目前不行，误差高于平均模板。
- 加一个很轻的监督映射（前缀 -> NMF 权重）后，25% 和 50% 前缀略优于平均模板。
- 这个提升不大，但说明当前前缀对未来曲线有弱可预测性。
- 75% 前缀下没有稳定提升，可能因为剩余区间太短，且未来新变化稀疏。

未来新高峰预测：

目前这个指标不适合作主要结论。原因是不同前缀下正负样本严重不平衡，很多通道在后半段很少出现“超过已观察峰值 15% 的新峰”。因此准确率容易被多数类支配，F1 也不稳定。

## 可解释性判断

`nmf_waveform_bases.png` 显示出若干可解释形态：

- 首段规划峰：思考规划波在最初阶段强烈抬升。
- 中段规划峰：思考规划波在 0.3-0.6 区间出现局部峰。
- 后段规划峰：思考规划波在 0.8-0.95 区间出现局部峰。
- 稳态输出/反馈基：模型输出波和环境反馈波在全程保持较平滑的背景形态。
- 复合后段基：后段输出、反馈和规划同时出现局部变化。

这支持“轨迹不是单峰，而是多个局部波形基叠加”的建模方向。

## 当前结论

当前数据用这个模型方法已经表现出：

| 问题 | 初步结论 |
|---|---|
| 是否有可解释性 | 有。NMF 学出的非负波形基能被解释成首段规划、中段规划、后段规划、背景输出/反馈等形态。 |
| 是否有可预测性 | 有弱信号。纯数学前缀拟合不够好，但加入轻量监督映射后，25%/50% 前缀预测未来曲线略优于平均模板。 |
| 是否足以做在线追踪 | 可以做第一版“形态追踪”，但还不适合声称能准确预测剩余进度。 |
| 下一步最该优化什么 | 用更合理的预测目标，例如剩余轮数、未来测试/错误事件、阶段切换，而不是只预测完整曲线。 |

## 重要限制

1. 当前环境反馈波大多是工具返回，不是真人用户输入。
2. 工具调用次数几乎固定为每轮 1，不适合作主要波形。
3. 未来新高峰标签仍然粗糙，不能作为强证据。
4. 30,000 条轨迹是样本实验，不是全量 413k 轨迹级样本。
5. 当前 NMF 学的是归一化形状，不保留绝对 token 规模。

## 下一步

建议后续做三个改进：

1. 用全量或更大样本重跑 NMF，并比较 `components = 4, 8, 12, 16` 的重构误差和可解释性。
2. 用 NMF 权重作为特征，训练剩余轮数、未来测试、未来错误反馈、最终成功率的 LightGBM/岭回归基线。
3. 做在线相位估计：给定前 N 轮，估计当前属于哪些波形基、当前处在什么阶段、后续是否还会进入调试/验证波段。
